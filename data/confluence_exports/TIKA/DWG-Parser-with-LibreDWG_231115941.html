TIKA : DWG Parser with LibreDWG

Created by Dan Coldrick on окт 05, 2022

Tika 2.51 now allows the ability to parse DWG files using LibreDWG (https://www.gnu.org/software/libredwg/) and DWGRead

TIKA-1735 - Unsupported AutoCAD drawing version: AC1027 Resolved


To enable this functionality set it via the config file giving the config file the location of DWGRead

<properties>
    <parsers>
        <parser class="org.apache.tika.parser.dwg.DWGParser">
            <params>
				<!--Location DWG read can be found-->
                <param name="dwgReadExecutable" type="string">g:/libredwg-0.12.5-win64/dwgread.exe</param>
				<!--Timeout in milliseconds before the DWGRead process is aborted-->
                <param name="dwgReadTimeout" type="long">100000</param>
             </params>
        </parser>
    </parsers>
</properties>
