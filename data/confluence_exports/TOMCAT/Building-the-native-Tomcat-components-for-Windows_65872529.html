Apache Tomcat : Building the native Tomcat components for Windows

Created by Mark Thomas, last modified on сен 27, 2016

Tomcat provides a number of native components. For most operating systems, a source package is provided and end users are expected to build the binary for themselves. The exception is Windows where it is far less typical for a user to have the tools to hand to build a native component from source.

These pages describe the build process used create convenience binary releases for these components for Windows.

The binaries share a  common build environment. Each convenience binary has a dedicated page for its build instructions:

  • Building the Tomcat Native Connector binaries for Windows
  • Building the isapi_redirector.dll (mod_jk) binaries for Windows
