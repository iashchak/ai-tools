Hadoop : How to declare end of life

Created by Wei-Chiu Chuang, last modified on дек 22, 2020

To declare a Hadoop release line end of life (say, 2.9.x), please start a DISCUSS thread in the hadoop mailing lists: common-dev, hdfs-dev, yarn-dev and mapreduce-dev.

After the initial consensus, start an official VOTE thread. If the vote passes, do the following:


  1. send an ANNOUNCE message to common-dev, hdfs-dev, yarn-dev, mapreduce-dev, user and general declaring the release line EOL. Example
  2. Go to https://issues.apache.org/jira/plugins/servlet/project-config/HADOOP/administer-versions?status=unreleased to archive the unreleased release (for example, 2.9.3) on the Jira. Repeat this step for YARN, HDFS and MAPREDUCE.
  3. Update the the EOL release branches wiki.
