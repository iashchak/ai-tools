Created by Junping Du, last modified on окт 30, 2017

Release schedule

It has been over 1 year since last minor release 2.7.0, so we plan to release Hadoop 2.8.0 as soon as we can.

Per discussions on community threads, we will release current 2.8 branch rather than cutting off a new one from branch-2. The left patches on branch-2 will move to next minor release (2.9) by default.
