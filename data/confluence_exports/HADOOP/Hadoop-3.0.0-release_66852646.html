Created by Andrew Wang, last modified on дек 15, 2017

Hadoop 3.0.0 has been released! Bits are available on the official website.

Release schedule

For Hadoop 3, we are planning to "release early, release often" to quickly iterate on feedback collected from downstream projects.

To this end, we will be releasing a series of alpha and beta releases leading up to an eventual Hadoop 3.0.0 GA.

This is a planned release schedule. Future release dates are subject to change.
